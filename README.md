# 🛸 Antigravity IDE: The Autonomous Agent Command Center

Antigravity is a specialized operational framework designed to transform your IDE into a high-performance workspace for autonomous AI agents. By enforcing a strict "Project-Resource" pattern and providing standardized workflows, Antigravity enables AI to code, research, and validate tasks with minimal human friction.

**Mission**: Instructions for creating a global working directory for autonomous operation for Agents in Antigravity IDE.

---

## 🎯 What is Antigravity?

Most AI coding assistants lack **Environmental Context**. They don't know where your docs are, they don't have a reliable way to run tests, and they often struggle with complex cross-file refactorings.

Antigravity solves this by:
1.  **Enforcing Structure**: Separating Code (`~/Projects`) from Knowledge (`~/Agents/resources`).
2.  **Standardizing Workflows**: Providing a library of discrete, AI-executable mission protocols (e.g., `yolo-full`, `rrw`, `start-project`).
3.  **Bootstrap Reliability**: Offering "Absolute Zero" scripts to prepare a fresh machine for agentic operations in minutes.

## 📁 The Antigravity Hierarchy

When setup is complete, your machine will follow this premium organizational pattern:
- **`~/Projects/`**: The home for all development repositories.
- **`~/Agents/antigravity-ops/`**: This repository—the brain and command center.
- **`~/Agents/resources/`**: Project-specific vaults containing non-code artifacts, task logs, and local validation reports.
